# How to setup

1. Clone the Repository
   ```cmd
   git clone https://github.com/The-Ghost-Network/Ghost-Node
   ```
2. Install The Dependencies
   ```cmd
   npm i
   ```
3. Run
   ```cmd
   npm start
   ```
